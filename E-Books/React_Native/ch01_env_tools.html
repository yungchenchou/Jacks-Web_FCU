  • 本章主要介紹撰寫 React Native 所需環境及程式編輯器

開發環境設定

  • 步驟 1: 下載並安裝 NODE
  • 步驟 2: 執行下列指令安裝 expo-cli 
    npm install -g expo-cli
  • 步驟 3: 切換到你要撰寫程式的目錄，執行下列指令初始化 專案
    expo init MyProject
cd MyProject
  • 步驟 4: 要執行剛剛建立的 APP 可執行下列指令
    expo start // 你也可以用 npm start 指令可以得到同樣的結果
